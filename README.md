## Projects

## 1. Erasmus Mobility Trends Analysis
**Tools:** R, Excel  
**Methods:** Spatial autocorrelation analysis, descriptive statistics, data visualization

This project investigates patterns and trends in Erasmus student mobility data through statistical and spatial analysis. Using R, I applied spatial autocorrelation methods to explore geographical patterns and relationships within mobility data. 

The project also includes an Excel-based descriptive dashboard designed to summarize key trends and communicate findings through accessible visualizations.

**Key competencies demonstrated:**
- Statistical analysis and modelling
- Spatial data analysis
- Data visualization and reporting 
- Translating quantitative results into meaningful insights

## 2. Face Detection in Historical Newspapers
**Tools:** Python  
**Methods:** FaceNet, computer vision, convolutional neural networks, trend analysis

This project explores changes in the visual representation of people in historical newspapers over a 200-year period. A computer vision pipeline was developed to detect and analyze faces in digitized newspaper pages using FaceNet-based face detection methods.

The extracted data was transformed into structured information and used to investigate historical trends in visual representation over time.

**Key competencies demonstrated:**
- Data processing pipelines
- Computer vision and machine learning
- Large-scale dataset analysis
- Feature extraction and trend visualization
- Python-based analytical workflows

[View project →](./newspaper_face_trends_with_facenet)

## 3. Lego Brick Classification
**Tools:** Python  
**Methods:** Convolutional neural networks, transfer learning, model evaluation

This project focuses on image classification using Lego brick images. A baseline convolutional neural network (CNN) was developed and compared against a transfer learning approach using the VGG16 architecture.

The project evaluates model performance differences and explores how pretrained deep learning models can improve image classification tasks.

**Key competencies demonstrated:**
- Machine learning model development
- Deep learning and computer vision
- CNN architecture comparison
- Model evaluation and performance analysis
- Python-based machine learning workflows
